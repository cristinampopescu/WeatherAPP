# Weather App

A simple Expo-based React Native application that displays weather information using geolocation, navigation, and Redux state management.

Start the iOS version:

```sh
npx expo run:ios
```

Start the Android version:

```sh
npx expo run:android
```

Run a specific test file:

```sh
npm test -- __tests__/MyCity.test.tsx
```

Run all tests:

```sh
npm test
```
